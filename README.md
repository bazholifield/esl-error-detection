# ESL Error Detector

This project automatically detects and categorizes grammatical and lexical errors in learner English writing, and provides explanations and corrections.

## Dataset
- **OmniGEC (WikiEdits-MultiGEC)** — English subset, human-corrected edits from Wikipedia.

## Features
- Grammar, spelling, and lexical error detection
- Explanation module for learner feedback
- Modular data preprocessing and model pipeline
